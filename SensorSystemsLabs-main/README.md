# Sensor Systems Laboratories

## Description

This repository documents my laboratory work for the Sensor Systems course at Politecnico di Milano.
It contains practical implementations of sensor interfaces, signal conditioning circuits, and microcontroller programming (STM32) developed during the semester.

The goal of these projects was to bridge the gap between theoretical sensor models and real-world embedded systems, mastering hardware peripherals and communication protocols.

## Tools & Technologies

**Hardware:** STM32 Nucleo-F401RE Board, Sensor Expansion Board.

**Software:** STM32CubeIDE, C Language, HAL Library.

**Protocols:** UART, I2C, SPI.
